# Android Assignment

A collection of 4 Android apps built with Kotlin.

## Apps

### Q1 - Currency Converter
- Converts between 10 currencies
- Fixed exchange rates
- Clean UI with dropdowns

### Q2 - Media Player
- Audio playback with play/pause/stop
- Video playback with MediaController
- Status display for both players

### Q3 - Sensor App
- Accelerometer readings
- Gyroscope readings
- Magnetic field readings
- Shake detection

### Q4 - Camera/Gallery
- Take photos with camera
- Pick images from gallery
- Display selected images

## Tech Stack
- Language: Kotlin
- Min SDK: API 24
- IDE: Android Studio
